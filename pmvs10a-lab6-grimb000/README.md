## How to build
Рекомендации по сборке:
1. Установить Android Studio (актуальная стабильная версия).
2. Убедиться, что установлены Android SDK Platform 36 и Build Tools для API 36.
3. Использовать JDK 11 (в проектах target/source compatibility = 11).
4. Минимальная версия Android для запуска: API 24 (Android 7.0).
5. В Android Studio открыть нужный проект (например, Task77) и выполнить Gradle Sync.
6. Запустить на эмуляторе/устройстве Android 7.0+.
