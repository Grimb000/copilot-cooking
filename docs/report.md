# Отчет по лабораторной работе 4

## Задание 1. Виды тестов в Android Studio
1. `src/test/java/` — локальные модульные тесты, которые выполняются без Android-фреймворка и UI. `src/androidTest/java/` — инструментальные тесты, которые запускаются на устройстве/эмуляторе и имеют доступ к Android API и интерфейсу.
2. Тесты из `src/test/java/` выполняются в JVM. Тесты из `src/androidTest/java/` выполняются в среде Dalvik/ART на устройстве или эмуляторе.
3. Для `androidTest` нужен эмулятор/устройство, потому что тесты используют Android-фреймворк, ресурсы и UI, которые доступны только в среде исполнения Android.

Скриншот: открыть `docs/report.md` в IDE и показать этот раздел.
